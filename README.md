# portfolio-optimization-efficient-frontier
##  Objective

To determine optimal stock allocation using Efficient Frontier and evaluate whether optimization adds value over simple strategies like equal-weight portfolios.

---
Built a portfolio optimization framework using Efficient Frontier to study allocation behavior, impact of constraints, and limitations of mean-variance optimization under real market data.

##  Final Takeaway

- Unconstrained optimization leads to unrealistic concentration in a few stocks due to sensitivity to return estimates  
- Adding constraints improves diversification but reduces the Sharpe ratio, highlighting the trade-off between optimality and practicality  
- Equal-weight portfolio serves as a strong baseline and, in some cases, performs comparably due to its robustness against estimation error  
- Portfolio optimization should not be used blindly, as results are highly dependent on input assumptions and can lead to unstable allocations  

- Overall, Efficient Frontier is a useful framework for understanding risk-return trade-offs, but real-world portfolio construction requires constraints, judgment, and robustness checks.
